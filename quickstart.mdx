## Development Workflow

<Steps>
  <Step title="Clone the Repository">
    ```bash
    git clone https://github.com/mvalencia464/stokeleads.git
    cd stokeleads
    npm install
    ```
  </Step>
  <Step title="Set Environment Variables">
    Copy `.env.example` to `.env.local` and fill in your GoHighLevel API credentials:
    ```bash
    VITE_HIGHLEVEL_TOKEN=your_api_key
    VITE_HIGHLEVEL_LOCATION_ID=your_location_id
    ```
  </Step>
  <Step title="Start Development Server">
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) to see the app.
  </Step>
  <Step title="Make Changes">
    Edit files in `app/` or `components/` directories. Changes hot-reload automatically.
  </Step>
</Steps>
